## Diamond Price Prediction

Predicted diamond prices using multiple linear regression 
based on 9,000+ diamonds dataset from Kaggle.

## Methods
- Simple and multiple linear regression
- Variable selection: Backward AIC and VIF
- Model diagnostics: ANOVA, R², confidence intervals

## Key Findings
- Depth, clarity, and width are the most significant predictors
- 95% confidence interval for prediction: [5.84, 5.97]

## Tools
R, ggplot2
